# BookMarket(notice-board)
이 게시판은 어느정도 기능을 하지만 서점 페이지를 개발해서 같이 붙여 볼 생각입니다. 원래는 각각 다른 교재에서 다른 이유로 배운 것들이지만 붙여 놓으면 효용이 더 올라갈 거라고 판단 했습니다. 

## 🚀 데모
- Live: https://Jhoon1018.github.io/emotion-diary-new

## ✨ 주요 기능

- 게시글 목록 조회(페이징)
- 게시글 작성
- 게시글 상세 보기
- 게시글 수정
- 게시글 삭제

## 🛠 기술 스택
- Java
- Html
